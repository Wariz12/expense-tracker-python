# Expense Tracker App (Python + Tkinter)

This is a modern GUI-based Expense Tracker built using Python, Tkinter, and Matplotlib.  
It allows users to:

- Add expenses
- Delete expenses
- View expenses in a table
- Run category analysis
- Export expenses to CSV
- View data in charts (Week 3 feature)

## Features
✔ Add and manage expenses  
✔ Modern GUI design  
✔ Category-based analysis  
✔ CSV export  
✔ Clean folder structure
